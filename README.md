This is a **Streamlit web application** that predicts house prices based on features like **borough, block, lot, zip code, year built, lot area**, etc., using a trained **Decision Tree Regressor** model.

The app allows users to either **upload a CSV file** with multiple houses or **input details manually** for a single house.

---

## **How to Use**

### 1. Upload CSV
1. Click **Upload CSV** button.  
2. Select a CSV file with the required features.  
3. The app will display the first few rows and predict house prices.  

### 2. Manual Input
1. Enter values for features like **Borough X, Block, Lot**, etc.  
2. Click **Predict Single House** button.  
3. The app will display the predicted price immediately.
